SHOOTER GAME
Простая игра написанная на Python.



## Требования
Python 3.12+
pyGame 2.6.1
pyInstaller 6.21.0 (опционально для пересборки проекта)


## Сборка
1. выполнить `pyinstaller -F -w .\shooter_game.py`
2. переместить бинарный файл `shooter_game.exe` или `shooter_game` из папки `dist` в корень проекта
3. запустить его
